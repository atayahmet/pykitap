##GİT
**yüklemek için;**  
sudo apt-get install git  
sudo apt-get install vim(Vim bir metin düzenleyicidir.) 
 
- #mkdir git  
- #cd git    
- #git init
- #vim git.txt (bişeyler yazın kaydedin)
- #git add git.txt
- #git commit -m "yeni bir git.txt dosyası eklendi"  
- #git status
